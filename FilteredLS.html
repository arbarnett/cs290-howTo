Path Module

Similar to the filestream module (which you also need in this exercise), you will need access to the path module. Accessing this is done exactly like you did for the filestream module:

var path = require('path');

This module contains a method that will allow you to access the extension names of files within a directory. The user is to input the directory name and the extension name they want to filter by into the command line. In order to compare the directory's files' extension names with the user-inputted extenstion, make a variable using the command line input. You will need to concatenate a period before this value:

var directory = process.argv[2];
var extension = '.' + process.argv[3];

readdir() Method

Similar to the last exercise, the readdir() method requires that the first parameter be the directory name, and the second parameter be a callback function. This callback function has two parameters. The first is err, where any errors that occur while reading the directory will be passed, and the second is list, where the list of files within the directory will be passed. Now, within that callback function, you need to compare your extension variable to each of the files' extensions. This is done using one of the path module's built in methods: the extname(filename) method.

The final code will look like this:

var fs = require('fs');
var path = require('path');
var directory = process.argv[2];
var extension = '.' + process.argv[3];

fs.readdir(directory, function(err, list){
     if (err) {
           console.log(err);
     }
     list.forEach(function(filename){
          var ext = path.extname(filename);
          if(ext === extension) {
               console.log(filename);
          }
     });
});


Note that in this code, there is a callback function used in the foreach loop. This function is performd on every file that is in the list.
