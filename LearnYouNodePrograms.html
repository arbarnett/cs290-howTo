Learn You Node Programs

To help you learn how to navigate and use Node.js, there are provided practice exercises called, Learn You Node. While geared more towards an audience with web-programming experience, they are beneficial. This tutorial is geared to aid beginner web-programmers with the basic fundamentals covered in the early exercises provided on Learn You Node.

How to access these programs

*You complete these steps after you install node.js*

1) Start your terminal.

2) Type in: 

sudo chown -R $(whoami) ~/.npm

This will change the ownership of npm to you

3) Now type:

sudo npm install -g learnyounode

This will install the Learn You Node practice program

4) Now type:

learnyounode

5) Now you can get some practice!

Just follow the instructions provided to navigate through the exercises. Use this tutorial as a supplemental material to help explain the crucial fundamentals introduced in the early exercises, primarily asyncroneous callback functions and modules. Access the supplementary material for each exercise by clicking the link on the left that corresponds to the exercise you are working on.
