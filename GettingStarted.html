Getting Stared with Node.js

What is Node.js?

Node.js is a server-side interpreter of JavaScript. It is a program built to run Javascript on the server. Although it runs JavaScript, it is written in C, so it is capable of handling all the grunt work required of a server-side program.

Why use it?

The big appeal of Node.js is that there is no need to learn another language, such as PHP. JavaScript is all you need! Node.js is also fast and scalable, meaning that it can process requests quickly and efficiently.
