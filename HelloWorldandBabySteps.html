"Hello World" and "Baby Steps"

Simple Output

To complete this exercise, you need to know console.log().

console.log() is a debugging tool used to print information to the console, or in the case of completeing the Learn You Node exercises, the Terminal.

Web developers use this tool to check what values variables hold at various points in the program. For example, with lines of code as follows: 

var x = 4; 
console.log(x);

This will print '4' to the console.

Other times, it can be used to see when a program breaks, by printing statements when it enters a function or a different page.

In the "Hello World" exercise, you just need to print the phrase "Hello World" to the console.

Command Line Arguments

Command line arguments are values passed to your program through the command line. There are two important variables that track these values. The first is argc. This is an integer that represents the number of arguments passed on the command line. The second is an array called argv. The elements in this array are the values you passed on the command line.

To access the elements of argv, you use the process.argv variable

Let's look at an example:

If I type on the command line this following code: 

node programName.js Allie Barnett

The argv array holds the following values in its indexes:

process.argv[0]= "node"
process.argv[1]="programName.js"
process.argv[2]="Allie"
process.argv[3]="Barnett"

Note that the first two values in this array are "node", the program that will run the file, and "programName.js", the file you would like to run.

Taking what we learned about simple output and command line arguments, let's use the code from above to print out a greeting the user who enters their name on the command line:

var firstName = process.argv[2];
var lastName = process.argv[3];

console.log("Hello, " + firstName + " " + lastName);

On the command line, if you were to type:

node programName.js Allie Barnett

The program would run, and it would print:

Hello, Allie Barnett
